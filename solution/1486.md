# Problem
[Link](https://leetcode-cn.com/problems/xor-operation-in-an-array/)

# Solution
* 直接模拟
* 时间复杂度O(n)

# Code
```cpp
class Solution {
public:
    int xorOperation(int n, int start) {
        int ans = 0;
        for (int i = 0; i < n; ++i) {
            int val = start + 2 * i;
            ans ^= val;
        }
        return ans;
    }
};
```
